## 助けてください!(Help wanted!)
### 何を手伝えますか？(What can I help you with?)

- 誤字脱字の修正
- カテゴリ、情報、ツールの追加
- 各項目の説明(details)の追加
- etc

### 助けるときのルール(Rules)

1. issueは必ず立ててください。
2. AssigneesとLabelsを付けるのを忘れないでください。  
  2-1. Labelsは6種類あります。
    - site(サイト関係)
    - documentation(ドキュメント関係)
    - bug(バグ全般)
    - proposal(提案)
    - question(質問)
    - others(その他)
3. ブランチは`feature/issue-番号`でお願いします。e.g. `feature/issue-9`
4. コミットメッセージは自由です。
5. PRは`issueタイトル(#issue番号)`でお願いします。e.g. `README.mdでのtypo修正(#6)`
